# Customer Review Sentiment Analyzer

## Project Overview
Analyzed 568,000 Amazon food reviews using NLP and sentiment analysis.

## What I did
- Performed EDA on real Amazon review dataset
- Built sentiment classifier using TextBlob NLP
- Compared TextBlob predictions against star ratings
- Visualized insights using seaborn and matplotlib
- Generated word clouds for positive reviews

## Libraries Used
pandas, matplotlib, seaborn, textblob, wordcloud

## Dataset
Download from: https://www.kaggle.com/datasets/snap/amazon-fine-food-reviews

## Results
- 88% of reviews classified as Positive
- Negative reviews averaged 2.75 star rating
- Most common positive words: taste, love, great, good
